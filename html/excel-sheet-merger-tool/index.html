How To Use

1. Select your Excel files using the "Choose Files" button.

2. The tool will automatically merge all data from the files (and all their sheets). Data is merged up to the last row of the first column of every sheet.

3. The tool adds a new column with the name of each sheet for every merged row.

4. Click "Download Merged Excel" to save the new file.

The download will happen automatically and the file will be saved to your browser's default download location.

All processing is done securely in your browser.

Disclaimer: This is a client-side tool and all data processing happens locally. No data is sent to any external server or third-party services.
